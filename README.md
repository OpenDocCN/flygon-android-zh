## 下载

### Docker

```
docker pull apachecn0/freelearn-android-zh
docker run -tid -p <port>:80 apachecn0/freelearn-android-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-android-zh
freelearn-android-zh <port>
# 访问 http://localhost:{port} 查看文档
```
